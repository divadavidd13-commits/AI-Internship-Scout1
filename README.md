# AI Internship Scout

An AI-powered internship discovery system that searches, filters, verifies, and ranks internship opportunities.

## Features

- Search internships from LinkedIn
- Filter by location
- Verify opportunities using Gemini AI
- Rank internships based on relevance
- Save results automatically
- Email notifications

## Tech Stack

- Python
- Gemini API
- BeautifulSoup
- Requests

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
